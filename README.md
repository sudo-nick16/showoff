## Showoff 

> (4th rewrite of my project showoff)

A platform to showcase your projects and look for ideas for the next one.
Microservices:

| Responsibility | Name |
| -------------- | ------ |
| Handle user and auth | apex |
| Api Gateway | nimbus |
| Handle projects | stellar |
| Handle emails | catalyst |
